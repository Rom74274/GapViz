## 5. Justifications des permissions

À coller dans le champ correspondant à chaque permission.

### `downloads`
```
Pour intercepter le téléchargement du fichier CSV exporté par l'utilisateur depuis Ahrefs, Semrush ou SE Ranking, et l'envoyer automatiquement à l'API Star Gap. Sans cette permission, l'utilisateur devrait télécharger manuellement puis re-uploader le fichier.
```

### `storage`
```
Pour stocker temporairement le token de session d'import (UUID éphémère, expire en 10 minutes) entre l'app Star Gap et l'onglet de l'outil SEO. Le token sert uniquement à relier l'export CSV au bon projet Star Gap de l'utilisateur authentifié. Aucune donnée sensible n'est stockée.
```

### `tabs`
```
Pour identifier l'onglet de l'outil SEO (Ahrefs / Semrush / SE Ranking) afin d'y afficher le banner de statut de l'import, et pour fermer automatiquement l'onglet après un import réussi (UX).
```

### `activeTab`
```
Pour permettre au popup de l'extension d'afficher le statut de connexion au provider courant (Ahrefs / Semrush / SE Ranking) en fonction de l'URL de l'onglet actif.
```

### Host permission : `*://app.ahrefs.com/*`
```
Pour injecter le content-script sur la page Ahrefs Organic Keywords, capturer le token Star Gap passé en paramètre d'URL, afficher le banner de statut, et récupérer le CSV exporté avec les cookies de session de l'utilisateur.
```

### Host permission : `*://*.semrush.com/*`
```
Pour injecter le content-script sur la page Semrush Organic Positions, capturer le token Star Gap passé en paramètre d'URL, afficher le banner de statut, et récupérer le CSV exporté avec les cookies de session de l'utilisateur.
```

### Host permission : `*://*.seranking.com/*`
```
Pour injecter le content-script sur la page SE Ranking Competitive Research, capturer le token Star Gap passé en paramètre d'URL, afficher le banner de statut, et récupérer le CSV exporté avec les cookies de session de l'utilisateur.
```

### Host permission : `*://kngkvaqovdnysmqrvxtj.functions.supabase.co/*`
```
Pour envoyer le CSV intercepté à l'API Star Gap (Supabase Edge Function) qui le parse et l'enregistre dans le projet de l'utilisateur. C'est le seul endpoint backend auquel l'extension communique.
```
